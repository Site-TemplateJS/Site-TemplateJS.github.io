Welcome to SiteTemplateJS!

An easy website building framework that lays out the design of the site, all without any CSS or styling. 100% responsive, and dynamic. Made for quickly and easily building out the structure of a site, and allowing users to customize the site to their liking.

All you have to do to use this framework is to include the cdn <script src="https://site-templatejs.github.io/site-templatejs.min.js"></script> , and give each site element the proper classname

To get the proper navbar layout, set the div class="nav", and set the div or span containing the navbar links to class="nav-right"

To create the proper content area layout, set the div class="content"

For the best, responsive layout, enclose the content area and sidebars (if you add them) with div with class="container"

To create the proper footer layout, set the div class for the footer to class="footer"
